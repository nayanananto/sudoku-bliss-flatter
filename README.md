# 🧩 Sudoku Game (Flutter)

## 📌 1. Introduction

This project is a mobile **Sudoku game application** built to provide an engaging and interactive puzzle-solving experience.

Sudoku is a classic number-placement puzzle known for its simplicity and logical challenge. This project focuses on:

* building a user-friendly interface
* implementing efficient puzzle generation
* enhancing gameplay with tracking features

---

## 📚 2. Related Work / Concept

Since Sudoku is purely a number-based puzzle, the game allows users full flexibility in interaction:

* Users can input and modify numbers freely
* Cells can be cleared or revealed (hint system)
* A timer tracks how long a player takes to solve the puzzle
* Completion time acts as a measure of efficiency
* Game state can be saved and resumed later

---

## ✨ 3. Project Features

* **User-friendly Interface**
  Clean and intuitive design for smooth navigation

* **Puzzle Generation**
  Algorithm-based generation of Sudoku puzzles

* **Gameplay Enhancements**

  * Cell highlighting
  * Hint/reveal option

* **Time Tracking**
  Records time taken to complete puzzles

* **Save & Resume**
  Allows users to continue unfinished games
