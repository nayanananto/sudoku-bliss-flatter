# 🧩 Sudoku Bliss (Flutter)

A simple Sudoku game built using **Flutter** and **Dart**.
This project was created as an early exploration of mobile app development and basic game logic.

---

## ✨ Overview

This application implements the core mechanics of a Sudoku puzzle, allowing users to interact with a 9×9 grid and input numbers while following standard Sudoku rules.

The focus of the project is on:

* understanding Flutter UI components
* handling user input
* implementing basic game logic

---

## 🛠️ Technologies

* Flutter
* Dart

---


## 🚀 Purpose

This project serves as:

* a learning exercise in Flutter development
* a demonstration of implementing a simple game logic
* an early portfolio project
